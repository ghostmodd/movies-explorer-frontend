# Верстка для проекта "Movies Explorer"
Проект содержит страницы:
* Главная,
* Фильмы,
* Сохраненные фильмы,
* Аутентификация,
* Навигация в виде "бургер-меню",
* Редактирование профиля.

*Использованные технологии:*
* HTML
* CSS
* React JS
